
## Initial steps:

1. Open GIT folder and then double click on the `bin` folder. Go inside: `C:\Program Files\Git\bin`

2. Inside that folder you will see: `git.exe`

3. Click the address bar and copy the full path, it should be: `C:\Program Files\Git\bin\git.exe`

![[Pasted image 20260214002113.png]]

## Now go back to Obsidian

Open:

`Settings → Community Plugins → Git`

In:

`Custom Git binary path`

Paste:

`"C:\Program Files\Git\bin\git.exe"`


---

# Then VERY IMPORTANT:

Close Obsidian completely  
Open it again.

- You must `Git:Pull` first before start working in a device. Therefore, you have to turn on `Auto Pull on startup` feature in the Obsidian Git Plugin.

- Change the ==Pull== settings and ==Commit-and-Sync== settings like this:

- ![[Pasted image 20260214083143.png]]

- Auto-commit and sync can be turned upon requirement, suggested to follow manual commit and sync, till you get a hang of this plugin.
